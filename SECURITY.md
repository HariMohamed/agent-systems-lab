# Security Policy

This repository contains AI agent skills and workflow material, including imported content that may be unreviewed.

Please report:

- unsafe skill commands
- prompt-injection risks
- secret, credential, token, or environment-variable exposure
- malicious or suspicious scripts
- supply-chain risks
- unauthorized network, filesystem, browser, database, deployment, messaging, or Git actions
- instructions that bypass access controls or redefine instruction priority

Do not include secrets or private credentials in reports.

Use the repository host's private security reporting feature when available. If private reporting is not available, open a public issue with only non-sensitive details and offer to provide sensitive reproduction details through a private channel.

Unreviewed skills must be treated as untrusted until maintainers complete safety, quality, and licensing review.
