# Import-Handoff Contract

Minimum evidence before proposing an import:
* candidate ID
* authoritative immutable source
* content hash
* license conclusion
* attribution requirements
* security-triage result
* overlap result
* compatibility result
* decision rationale
* proposed destination
* explicit human approval

State:
Candidate approval authorizes only a proposed import into incubating.
It does not authorize installation, execution, approval, publication, deployment, or global configuration.
