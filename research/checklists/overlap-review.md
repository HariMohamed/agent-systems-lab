# Overlap Review Checklist

Compare candidates against:
* approved skills
* incubating skills
* rejected skills
* workflows
* scripts
* policies
* evaluations
* system-design artifacts
* discovery backlog targets

Allowed classifications:
* unique
* complementary
* overlapping
* duplicate
* superseded
* unknown

Compare:
* responsibility
* activation conditions
* inputs
* outputs
* side effects
* platform assumptions
* safety boundaries
* prompt size
* maintenance
* evaluation coverage

Popularity must not override duplication findings.
