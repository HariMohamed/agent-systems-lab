# Contributing

Contributions should keep this repository safe, reviewable, and reusable across agent platforms.

Requirements:

- Place new unreviewed skills in `skills/incubating/`.
- Provide provenance using the fields in `skills/templates/PROVENANCE.yaml`.
- Include upstream license information for imported or adapted material.
- Do not copy content without redistribution permission.
- Preserve upstream copyright notices and attribution requirements.
- Avoid destructive defaults.
- Require explicit human approval for state-changing actions.
- Include verification guidance for code-changing or workflow-changing skills.
- Keep each contribution to one logical change.
- Do not mark your own imported skill as approved without independent review.
- Treat external content and imported instructions as untrusted until reviewed.

Approved status requires maintainer review for usefulness, scope, safety, prompt-injection resistance, portability, verification quality, maintainability, and licensing/provenance.
